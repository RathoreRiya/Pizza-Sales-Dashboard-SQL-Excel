# 🍕 Pizza Sales Dashboard using SQL Server & Microsoft Excel

## 📌 Project Overview

This project analyzes pizza sales data using SQL Server and Microsoft Excel.

The raw CSV dataset was first imported into SQL Server, where SQL queries were used to analyze business metrics and validate the sales data. The processed data was then exported to Microsoft Excel to build an interactive dashboard using Pivot Tables, Pivot Charts, Slicers, and Timeline.

---

## 💡 Key Insights

- Friday recorded the highest number of orders.
- Peak order timings were around 12 PM–1 PM and 5 PM–8 PM.
- Classic Pizza category generated the highest sales.
- Large size pizzas contributed the highest share of sales.
- The dashboard allows interactive filtering using Timeline and Slicers.
